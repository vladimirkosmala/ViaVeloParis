ViaVeloParis
============

Map of available Velib' (bicycles for rent) in Paris for iOS.

It uses www.velib.paris.fr API for realtime information about bikes, Google Maps API for calculating distances.

Usage
=========

- Open the project on Xcode
- Build and play
